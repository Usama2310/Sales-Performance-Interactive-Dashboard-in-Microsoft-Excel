Sales Performance Interactive Dashboard

Overview
This project is an interactive sales dashboard created in Microsoft Excel to analyze sales performance and visualize key metrics. It provides insights into total sales, employee performance, and target achievement using dynamic filters and charts.

How to Use

Step 1 — Download the File
Download the Excel file to your computer.

Step 2 — Enable Macros
Since this file contains macros, you must enable them before using the dashboard.

Important
After downloading the file, macros may be blocked by Excel for security reasons.

To unlock macros

Close the Excel file if it is open.

Right click the downloaded file.

Click Properties.

At the bottom of the General tab, check the box that says Unlock.

Click Apply and then OK.

Open the file again and click Enable Content if prompted.

Step 3 — Navigate the Dashboard
Go to the Dashboard sheet to interact with the visuals and filters.
Use the Region slicer to filter data dynamically.

Features
Interactive dashboard with dynamic filtering
Sales performance insights
Target achievement analysis
Professional layout and visuals

Requirements
Microsoft Excel with macros enabled

Purpose
This project is created for learning and portfolio purposes to demonstrate data analysis and dashboard development skills.
